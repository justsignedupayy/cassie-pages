About CASSIE

CASSIE (pronounced kæsiː) is a comprehensive cloud-based platform designed to revolutionize genomic data analysis. Our platform provides researchers and bioinformaticians with powerful tools to assemble and annotate genomes efficiently, reducing the complexity and time required for genomic research.

Project Overview

This senior design project brings together a team of talented developers to create a cutting-edge solution for the genomics community. CASSIE addresses the growing need for accessible, efficient, and powerful genome assembly tools in the era of big data genomics.

Our platform combines modern web technologies with robust bioinformatics algorithms to deliver a comprehensive solution that serves both researchers and institutions working with genomic data.
