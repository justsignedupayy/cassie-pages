About CASSIE

CASSIE is a cloud-native genomics workflow platform that replaces manual pipeline construction with guided UI selections. User-defined configurations are translated into reproducible Nextflow workflows orchestrated on scalable Kubernetes infrastructure.

Workflow Tools

Key Features

Guided tools for reproducible, cloud-native genomics workflows

Guided Workflow Setup

Replace manual pipeline construction with guided UI selections, smart suggestions, and reusable pipeline options.

Nextflow Reproducibility

User-defined configurations are translated into reproducible Nextflow workflows for consistent analysis runs.

Scalable Cloud Computation

Workflows run on Kubernetes-backed cloud infrastructure built for compute and memory intensive genomics workloads.

Centralized Data Management

Upload genomic files from your computer or Google Drive and manage workflow inputs and outputs in one place.

Resource & Cost Estimation

Estimate runtime and cost from input size, selected tools, and compute resource needs before execution.

Community Pipelines

Use shared pipelines and discussions to support reproducibility, reuse, and smoother team collaboration.

Project Context

Project Overview

CASSIE addresses the need for accessible, reproducible, and scalable genomics workflows. It helps users move from uploaded data to configured pipelines, estimated resources, cloud execution, and comprehensive results.

The platform combines guided UI selections with reproducible Nextflow workflows orchestrated on Kubernetes, reducing manual setup while preserving workflow clarity and repeatability.
